# Icon Requirements

Run `npm run setup` to generate placeholder icons.

Replace before shipping:
- build/creatorradar-icon.png  (256x256 PNG)
- build/creatorradar-icon.ico  (multi-size Windows ICO: 16,32,48,256)
- build/installer-banner.bmp   (150x57 BMP)
- build/installer-sidebar.bmp  (164x314 BMP)

Free PNG→ICO: https://convertico.com
